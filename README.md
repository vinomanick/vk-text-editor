# vk-text-editor

## Installation

```bash
npm install vk-text-editor --save
```

## Usage

#### 1. Require and use sample react components inside your application.

### Button component:

```js
<VkButton
  kind = 'primary'
  id = 'submitBtn'
  type = 'submit'
  disabled = true
  onClick = {this.onClick},
  label='Submit'
/>
```

### Badge component

```js
<VkBadge
  value = 10
/>
```
